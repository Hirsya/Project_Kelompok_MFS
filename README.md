# Project_Kelompok_MFS
project praktikum alpro 2026
bagi kalian yang mau bikin branch baru gunakan code dibawah ini pada cmd / Terminal (pastikan sudah menginstall git pada pc masing"):
1. buat folder baru (contoh : dev-hirsya)
2. ketik cmd pada search folder kalian setelah membuka folder yang baru dibuat di file manager (kalau bingung nanya di grup)
3. setelah buka cmd gunakan kode berikut :
4. git init
5. git clone https://github.com/Hirsya/Project_Kelompok_MFS.git
6. cd Project_Kelompok_MFS
7. git checkout -b <nama branch kalian> (contoh : git checkout -b dev-hirsya)
8. git push -u origin <nama branch kalian> (contoh : git push -u origin dev-hirsya)
9. selamat sekarang edit kode di branch masing"
